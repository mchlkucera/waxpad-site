I loved Raycast Notes. Fast, minimal, always one keystroke away.

But my notes were locked inside Raycast. My AI agent couldn't see them. No export. No local files.

So I built Waxpad. Same floating window. Same markdown. But your notes are just .md files in ~/.waxpad/ — and any tool can read them.

~/.waxpad/*.md

Plain markdown files on disk. Claude, Cursor, grep — they all just work.

Option + N

Summon the window from anywhere. It floats above everything.

Live markdown

Headings, checkboxes, code blocks — rendered as you type.
